## 📋 ПОДГОТОВКА К ДЕПЛОЮ

### 1. Установка переменных окружения

**НЕ КОММИТЬТЕ ЭТИ ПЕРЕМЕННЫЕ В GIT!**

Создайте файл `.env.deploy` (добавьте его в `.gitignore`):

```bash
# IP адрес сервера
export DEPLOY_SERVER_IP="188.120.231.71"

# Пользователь для SSH
export DEPLOY_SERVER_USER="root"

# Пароль сервера (лучше использовать SSH ключи!)
export DEPLOY_SERVER_PASS="ваш_безопасный_пароль"

# Токен Telegram бота
export DEPLOY_BOT_TOKEN="ваш_токен_бота"

# Домен (опционально, по умолчанию astrogame-io.ru)
export DEPLOY_DOMAIN="astrogame-io.ru"

# Путь к SSH ключу (рекомендуется вместо пароля)
# export DEPLOY_SSH_KEY="$HOME/.ssh/id_rsa"
```

### 2. Загрузка переменных

```bash
source .env.deploy
```

## 🔑 ИСПОЛЬЗОВАНИЕ SSH КЛЮЧЕЙ (РЕКОМЕНДУЕТСЯ)

### 1. Генерация SSH ключа

```bash
ssh-keygen -t ed25519 -C "deploy@casino" -f ~/.ssh/casino_deploy
```

### 2. Копирование ключа на сервер

```bash
ssh-copy-id -i ~/.ssh/casino_deploy.pub root@188.120.231.71
```

### 3. Использование в скрипте

```bash
export DEPLOY_SSH_KEY="$HOME/.ssh/casino_deploy"
export DEPLOY_SERVER_IP="188.120.231.71"
export DEPLOY_SERVER_USER="root"
# DEPLOY_SERVER_PASS не нужен при использовании SSH ключей

bash deploy-full.sh.safe
```
